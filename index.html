1
Sign up at railway.app
Go to railway.app → Sign up with GitHub. You get $5/mo free credit (no credit card needed to start).
2
Push 3 files to a GitHub repo
Your GitHub repo needs exactly these files:
📄 server.py
📄 requirements.txt
📄 Procfile (see step 3)
3
Create a Procfile in your repo
Create a file named Procfile (no extension) with this content:
web: gunicorn server:app --bind 0.0.0.0:$PORT --timeout 120 --workers 2
💡 Railway reads the Procfile to know how to start your app
4
Deploy from GitHub on Railway
On Railway dashboard → New Project → Deploy from GitHub repo → select your repo → Railway auto-deploys. Takes ~2 min.
💡 Watch the Deploy Logs — look for Booting worker to confirm it's running
5
Get your Railway URL
In Railway → your service → Settings → Networking → click Generate Domain. Your URL will look like:
https://mailcheck-backend-production.up.railway.app
💡 Railway stays awake 24/7 — no cold-start delays like Render free tier!
6
Connect the frontend
Click ⚙ (top-right) → paste your Railway URL → Test connection → Save & Connect.

Python backend is not running
Follow these steps one at a time in PowerShell or Terminal
1
Open PowerShell or Terminal
Press Win + R, type powershell, press Enter. Or search "PowerShell" in the Start menu.
2
Navigate to the backend folder
Type this command and press Enter. Replace the path with wherever you saved the mailcheck-backend folder.
cd "C:\Users\YourName\Downloads\mailcheck-backend"
💡 Tip: In Windows Explorer, hold Shift and right-click the folder → "Open PowerShell window here"
3
Install required Python packages
Run this command and wait for it to finish. You only need to do this once.
pip install flask flask-cors dnspython
💡 If "pip" is not found, try: python -m pip install flask flask-cors dnspython
4
Start the backend server
Run this command. You should see "Running on http://localhost:5000" — leave this window open while using MailCheck.
python server.py
💡 The server must stay running. Don't close the PowerShell window while verifying emails.
5
Refresh this page
Once the server is running, click the retry button in the top-right corner or refresh the page. The status dot will turn green.
